# 🍽️ Orrderlo Modern Restaurants

A modern, premium restaurant landing page built as a frontend assignment inspired by **Orrderlo**. The project focuses on creating a high-end SaaS-style user experience with beautiful animations, responsive layouts, and modern UI/UX principles. It showcases how a restaurant or food business can present its brand through an elegant digital experience.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/arafatgw5-cpu/Orrderlo-Modern-Restaurants.git
```

Go to the project folder:

```bash
cd Orrderlo-Modern-Restaurants
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Run production:

```bash
npm start
```
